# Personal Portfolio Website

## 🌟 Overview

This is my personal portfolio website built using React, showcasing my professional work, skills, and experience. I designed this site to provide a comprehensive and interactive view of my professional journey.

**Live URL**: https://portfolio-nurus-projects-d98f949e.vercel.app

## 🚀 Technologies Used

- React
- React Router
- State management library: Context API
- Styling framework: Tailwind CSS, Material-UI
- Animation libraries: Framer Motion

## ✨ Features

- Responsive design
- Interactive project showcases
- Detailed about me page
- Contact page

## 🛠 Installation

### Prerequisites

- Node.js (v14.0.0 or later)
- npm or Yarn

### Setup Steps

1. Clone the repository

   ```bash
   git clone git@github.com:nuru484/portfolio.git
   ```

2. Navigate to the project directory

   ```bash
   cd portfolio
   ```

3. Install dependencies

   ```bash
   npm install
   # or
   yarn install
   ```

4. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```
